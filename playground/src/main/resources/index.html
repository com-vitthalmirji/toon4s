üí° What is TOON?

TOON (Token-Oriented Object Notation) is a compact, LLM-friendly data format that saves 30-60% on token costs compared to JSON while maintaining full compatibility.

It combines YAML-style indentation with CSV-like tabular efficiency to create a format that's both human-readable and optimized for AI applications.
